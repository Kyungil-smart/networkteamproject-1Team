# PlayerSpawnManager 팀 배정

## 목적
- `SpawnAllPlayers` 단계에서 로드 완료한 플레이어를 `A(시민)` / `B(마피아)` 팀으로 나눈다.
- `B` 팀 인원은 `_targetTeamBCount` 필드로 고정한다.

## 동작 방식
1. 스폰시 `clientsCompleted` 목록을 복사한다.
2. 복사한 목록을 랜덤 셔플한다.
3. 셔플된 순서에서 앞의 `_targetTeamBCount`명만 `B`로 배정한다.
4. 나머지 인원은 모두 `A`로 배정한다.

## 참고
- 외부에서는 `TryGetAssignedTeam`으로 배정 결과를 조회할 수 있다.
