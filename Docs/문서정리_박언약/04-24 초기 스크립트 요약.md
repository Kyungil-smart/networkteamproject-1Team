# 04-24 초기 스크립트 요약
Scripts 폴더에 옮긴 초기 스크립트와 Input 폴더의 인풋시스템 관련 간단한 내용

## 📁 Core
- **`LinkManager.cs`**
  실행 시 자동으로 생성되는 싱글톤 매니저로, 서버와 연결이 끊길 시 메인 씬으로 돌아오는 코드가 있습니다
- **`RelayNetworkService.cs`**
  static 클래스로, Relay 초기화, Host/Client 접속 코드가 있습니다

## 📁 Model/Lobby (로비 및 세션 기능)
- **`NetworkBootstrap.cs`**
  UI 와 연동되어, `RelayNetworkService`를 호출해 네트워크 접속합니다
- **`SessionLauncher.cs`**
  로비에서 인원이 모였을 때, 호스트가 특정 입력(1번)으로 인게임 씬으로 전환하는 코드가 있습니다

## 📁 Model/InGameRule (인게임 공통 규칙)
- **`PlayerSpawnManager.cs`**
  모든 클라이언트가 씬 로드를 완료하면, 스폰 포인트들에 플레이어 프리팹을 생성(Spawn)해 줍니다

## 📁 Settings/Input (입력 시스템 관리)
- **`BattleInputReader.cs`** (`Assets/Settings/Input/`)
  Generate C# 방식의 Input System 입력을 받아 외부 이벤트를 실행하는 SO입니다. 여러 스크립트가 이 SO를 참조하여 입력을 처리하게 돕습니다.
